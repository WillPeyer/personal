Harmonize - CS 309 Team Project

As part of a semester-long project in CS 309, I contributed to the development of an innovative social networking application designed to connect users based on their musical preferences. Our team's efforts were recognized with a second-place finish among approximately 50 competing groups.

My Role and Contributions

I took primary responsibility for a significant portion of the frontend development, utilizing Java for programming and XML for page markup. Key features I implemented include:

  • Live activity feed
  • Friend management system (add/remove functionality)
  • Song search capability, leveraging the Spotify API
  • Personal music library management (add/remove songs from liked list)
  • Song snippet playback feature for recommended songs on the home feed

Technical Skills Acquired

This project significantly enhanced my technical skillset, providing hands-on experience with:

  • Data Transfer Objects (DTOs)
  • JSON Web Tokens (JWTs)
  • WebSocket technology
  • Comprehensive software development lifecycle
  • Agile methodologies and team collaboration
  • Integration with third-party APIs (Spotify)
  • Audio playback implementation in mobile applications

Outcome and Reflection

Our app's success in the class competition underscores the effectiveness of our collaborative efforts and the quality of our implementation. The integration of the Spotify API and the song snippet feature not only enhanced the user experience but also provided valuable insights into working with external services and handling media playback. This project not only honed my technical skills but also emphasized the critical importance of cohesive teamwork in software development. I take great pride in our collective achievement and the valuable learning experience it provided.
